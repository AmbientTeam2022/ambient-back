# inner enableCors

Habilita CORS (Cross-Origin Resource Sharing) indicando cuáles son las direcciones de confianza desde donde se originarán las solicitudes. Cualquier solicitud proveniente de otras direcciones IP serán rechazadas. Esto solo aplica a los endpoints de la API, y pueden añadirse excepciones.
