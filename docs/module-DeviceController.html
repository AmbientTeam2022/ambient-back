# async static updateSensors()

PATCH /device/ Obtiene los datos de un dispositivo

Endpoint dirigido al dispositivo Arduino para que actualice los datos de sus sensores.

Recibe el UUID y la Contraseña del dispositivo. Si las credenciales no son correctas no hace ningún cambio. Solo actualiza los valores de los sensores ya definido, de modo que no permite agregar o quitar sensores a través de este endpoint.
